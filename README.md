# INFO4607_Project
ProLifiq. A web based application to help optimize student lifestyle with features such as an interactive calendar that students can import their class schedule to, dining hall meals, and events for career building or socialization around campus. This applciation will contain a productivity heatmap and scores to help motivate users to stay on top of their day to day tasks to make the mundane tasks in life more engaging. Features like a financial planner will be implemented to help students build a foundation that they can carry beyond academia. This will also have an all encompassing LLM to help tie everything together to help keep the user on top of their day to day tasks. Students will be able to share calendars with housemates, friends, or group project partners to keep all on track or motivated or even a new way you can compete with your friends in a constructive way.
